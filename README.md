# Personal Portfolio Website

A modern, responsive portfolio website built with a focus on clean design and smooth animations.

## Features

- Responsive navigation with smooth scrolling
- Dynamic network background animation
- Typewriter text effect
- Interactive timeline for experience/education
- Skills showcase with hover effects
- Project grid layout
- Contact section with social links
- Dark theme optimized design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Inter, Roboto Mono)

## Project Structure

```
/boxxxi.github.io
│── index.html           # Main HTML file
│── style.css            # Global CSS for styling
│── script.js            # JavaScript for interactivity and animations
│── assets/              
│   ├── images/          # Store profile pictures, logos, etc.
│   ├── icons/           # SVGs or font icons (if any)
│   ├── fonts/           # Custom fonts (if used)

│   ├── about.html       # About me section
│   ├── projects.html    # Projects section

│── css/                 
│   └──glassmorphism.css  # Styles for the glass effect
│── js/                  
│   ├── animations.js     # Handles animations and transitions
│   └──theme-toggle.js   # If you add dark/light mode
│── README.md            # Documentation for your repo
└── .gitignore           # Ignore unnecessary files
```
