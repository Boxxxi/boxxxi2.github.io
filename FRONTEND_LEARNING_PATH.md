# Frontend Development Learning Path

## Introduction
This README outlines a structured learning path to help you understand and extend the frontend code of your portfolio website. By working through these exercises, you'll gain practical experience with HTML, CSS, and JavaScript while improving your website.

## Getting Started

### 1. Understanding the Structure (Week 1)
- **Task**: Map out the entire website structure
- **Exercise**: Create a visual diagram showing how HTML files, CSS files, and JavaScript files are connected
- **Goal**:  Understand how the components work together

### 2. CSS Fundamentals (Week 1-2)
- **Task**: Understand the CSS organization
- **Exercises**:
  - Identify and document all CSS variables in `variables.css`
  - Change the color scheme by modifying only the CSS variables
  - Create a new theme option (light/dark mode alternative)
- **Goal**: Learn how CSS variables create a consistent design system

## Intermediate Challenges

### 3. Responsive Design (Week 2-3)
- **Task**: Understand mobile responsiveness
- **Exercises**:
  - Review all media queries and document their purpose
  - Create a new breakpoint for ultra-wide screens (>1800px)
  - Optimize one section of your choice for different device orientations
- **Goal**: Master responsive design principles

### 4. CSS Animations (Week 3-4)
- **Task**: Understand and extend animations
- **Exercises**:
  - Document all keyframe animations in the codebase
  - Modify the glow effect on headings
  - Create a new entrance animation for project cards
  - Implement a custom loading animation
- **Goal**: Learn CSS animation techniques

### 5. JavaScript Interactivity (Week 4-5)
- **Task**: Understand the JavaScript functionality
- **Exercises**:
  - Document all event listeners in the codebase
  - Extend the typewriter effect to work with multiple text strings
  - Add a "scroll to top" button that appears after scrolling down
  - Implement smooth scrolling for navigation links
- **Goal**: Learn how JavaScript adds interactivity

## Advanced Projects

### 6. Dynamic Content Loading (Week 5-6)
- **Task**: Understand how content is loaded dynamically
- **Exercises**:
  - Study the section-loader.js file
  - Create a new section that loads content from a JSON file
  - Add a filter system for your projects section
- **Goal**: Learn dynamic content generation

### 7. Advanced Animations (Week 6-7)
- **Task**: Create complex animations
- **Exercises**:
  - Implement a parallax scrolling effect for the hero section
  - Create an interactive animation that responds to mouse movement
  - Add a 3D card flip effect to project cards
- **Goal**: Master advanced CSS and JavaScript animations

### 8. Performance Optimization (Week 7-8)
- **Task**: Optimize website performance
- **Exercises**:
  - Audit your website using Lighthouse
  - Implement lazy loading for images
  - Optimize CSS delivery (critical CSS)
  - Reduce JavaScript execution time
- **Goal**: Learn web performance optimization techniques

## Final Projects

### 9. Custom Feature Implementation (Week 8-9)
- **Task**: Add a completely new feature
- **Ideas**:
  - Interactive timeline with filtering
  - Dark/light mode toggle with animation
  - Custom cursor effects
  - Interactive skill visualization
- **Goal**: Apply everything you've learned to create something unique

### 10. Documentation and Article (Week 9-10)
- **Task**: Document your learning journey
- **Exercises**:
  - Create detailed documentation of your website's architecture
  - Write an article about learning frontend with AI assistance
  - Include before/after comparisons
  - Share challenges and solutions
- **Goal**: Solidify your knowledge by teaching others

## Specific Component Exercises

### Masonry Gallery
- Study how the masonry gallery works
- Extend it with filtering capabilities
- Add a lightbox feature for viewing full-size images

### Typewriter Effect
- Understand how the typewriter animation is implemented
- Extend it to cycle through multiple phrases
- Add a blinking cursor that changes color

### Glassmorphism Effects
- Study the glassmorphism CSS
- Create a new component using this effect
- Experiment with different blur and transparency values

### Interactive Map
- Understand how the world map is implemented
- Add new interaction features (zoom, pan)
- Create a custom visualization for a different dataset
